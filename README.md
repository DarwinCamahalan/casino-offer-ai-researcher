## ğŸš€ Getting Started

### What You'll Need
- A computer with internet connection
- Node.js installed (version 18 or higher)
- An OpenAI API key (get one from [OpenAI's website](https://openai.com))

### Quick Start

1. **Download the project**
   ```bash
   git clone https://github.com/DarwinCamahalan/casino-offer-ai-researcher.git
   cd casino-offer-ai-researcher
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up your environment**
   
   Create a `.env.local` file and add:
   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   NEXT_PUBLIC_XANO_API_BASE_URL=your_xano_api_url_here
   ```

4. **Start the application**
   ```bash
   npm run dev
   ```

5. **Open in your browser**
   
   Navigate to [http://localhost:3000](http://localhost:3000)
